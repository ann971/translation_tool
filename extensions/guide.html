🔑 如何找到 DeepL API Key

1確認你註冊的是「API 帳號」

DeepL 有兩種帳號：

  • DeepL 翻譯器帳號（一般用戶）→ 沒有 API Key
  • DeepL API 帳號（開發者用）→ 才有 API Key

你必須到 deepl.com/pro-api 申請 API 方案（有免費版）。

2找到 API Key 的位置

登入後，前往以下其中一個路徑：

方式網址
直接進入 API Keys 頁面 deepl.com/your-account/keys
帳號摘要頁 deepl.com/pro-account/summary

API Key 在登入後的「API Keys & Limits」分頁中可以找到與管理。

3複製 Key

基於安全考量，完整的 API Key 不會直接顯示在列表中，需要點擊「Copy」按鈕將 Key 複製到剪貼簿。

⚠️ 常見問題：為什麼看不到 Key？
  1. 用錯帳號類型：你可能註冊的是一般翻譯帳號，不是 API 帳號，需要另外申請 API 方案
  2. 還沒建立 Key：你可以點擊「Create new key」來建立一組新的 API Key，建立後會跳出視窗讓你複製
  3. 需要信用卡驗證：即使是免費方案，也需要輸入信用卡資料（僅作身份驗證用途）
