DeepL Select-to-Translate 安裝與使用教學
==========================================

【事前準備】

  1. 使用 Chrome 瀏覽器（或 Edge、Arc 等 Chromium 核心瀏覽器）
  2. 申請 DeepL API Key（免費方案即可）：
     https://www.deepl.com/your-account/keys


【安裝步驟】

  1. 下載本資料夾（extensions）到電腦中的任意位置
     ※ 安裝完成後請勿移動或刪除此資料夾，否則擴充功能會失效

  2. 開啟 Chrome，在網址列輸入：
     chrome://extensions/

  3. 開啟右上角的「開發人員模式」開關

  4. 點擊左上角的「載入未封裝項目」按鈕

  5. 選取剛才下載的 extensions 資料夾

  6. 安裝完成，工具列會出現擴充功能圖示


【設定】

  1. 在擴充功能圖示上按右鍵 → 選擇「選項」（或在 chrome://extensions/ 找到本擴充功能，點擊「詳細資料」→「擴充功能選項」）

  2. 輸入你的 DeepL API Key

  3. 選擇目標語言（預設為繁體中文）

  4. 如果你有 DeepL Pro 付費方案，開啟 DeepL Pro 開關

  5. 點擊「儲存設定」


【使用方式】

  ● 句子翻譯
    在任意網頁選取一段文字 → 文字右上方會出現藍色小點
    → 滑鼠移入小點或點擊 → 顯示翻譯結果

  ● 單字翻譯
    選取單一個單字 → 同樣出現藍色小點
    → 觸發後會顯示該單字的翻譯、音標、詞性及各項釋義

  ● 關閉面板
    按 Esc 鍵，或點擊面板外的空白處

  ● 移動面板
    拖曳面板頂部的標題列即可移動位置

  ※ 當選取的文字與目標語言屬於同語系時（例如目標語言為繁體中文，
     選取的是中文內容），不會出現藍色小點，也不會觸發翻譯。


【常見問題】

  Q: 出現「尚未設定 DeepL API Key」？
  A: 請到擴充功能選項頁面設定你的 API Key。

  Q: 出現「逾時：背景程式未回應」？
  A: 請重新整理網頁，或到 chrome://extensions/ 點擊本擴充功能的重新整理按鈕。

  Q: Chrome 啟動時出現「停用開發人員模式擴充功能」提示？
  A: 這是 Chrome 對手動載入的擴充功能的正常提醒，點擊取消即可繼續使用。

  Q: 更新版本後如何套用？
  A: 用新版檔案覆蓋原本的 extensions 資料夾，然後到 chrome://extensions/
     點擊本擴充功能的重新整理按鈕（圓形箭頭圖示）。
