# 翻譯插件下載頁面 — 產品需求規格書

## 1. 專案概述

### 目的
建立一個仿 Chrome Web Store 風格的翻譯瀏覽器插件下載頁面，提供插件介紹、下載及安裝教學。

### 目標使用者
一般大眾（所有需要網頁翻譯功能的使用者）

---

## 2. 插件功能說明

- **選取翻譯：** 使用者在網頁上選取文字後，選取文字右上方會出現一個藍色小點
- **觸發翻譯：** 游標移入藍色小點後，自動顯示翻譯結果
- **語言設定：** 使用者可在插件設定中選擇目標語言
- **自動偵測：** 自動偵測選取文字的語言，若與設定語言不同則進行翻譯

---

## 3. 網站內容架構

### 3.1 頂部區塊（Hero）
- 插件圖示（Logo）
- 插件名稱
- 簡短描述（一句話介紹）
- 下載按鈕（點擊下載 .zip 檔案）
- 基本資訊標籤（版本、大小、更新日期）

### 3.2 截圖展示區
- 插件操作截圖輪播（3～5 張）
- 展示選取文字 → 藍色小點 → 翻譯結果的完整流程

### 3.3 詳細描述區
- 功能介紹
- 特色亮點
- 支援語言列表

### 3.4 安裝教學區
- 步驟圖文說明：
  1. 下載 .zip 檔案並解壓縮
  2. 開啟 Chrome，前往 `chrome://extensions/`
  3. 開啟右上角「開發人員模式」
  4. 點擊「載入未封裝項目」
  5. 選取解壓縮後的資料夾
  6. 完成安裝，開始使用

### 3.5 附加資訊區
- 版本號
- 檔案大小
- 最後更新日期
- 語言支援

## 4. 視覺設計規範

### 色彩
- **主色調：** 藍色系（#4285F4 為主色，呼應插件藍色小點特色）
- **輔助色：** 淺灰背景（#F8F9FA）、深灰文字（#202124）
- **強調色：** 綠色用於下載按鈕（#1A73E8）

### 風格
- 簡潔現代，參考 Chrome Web Store 排版
- 圓角卡片式設計
- 充足的留白空間
- 清晰的資訊層級

### 字體
- 系統字體優先（-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto）

## 5. 技術規格

### 前端框架
- React

## 6. 頁面文案（初稿，後續細修）

### 插件名稱
Smart Translate

### 簡短描述
「輕鬆選取、即時翻譯 — 讓閱讀外語網頁不再是障礙」

### 詳細描述
Smart Translate 是一款輕量快速的瀏覽器翻譯插件。只需在網頁上選取文字，將游標移入右上角出現的藍色小點，即可立即獲得翻譯結果。

**主要特色：**
- 選取即翻譯，操作直覺不打斷閱讀
- 支援多國語言自動偵測
- 可自訂目標翻譯語言
- 輕量設計，不影響瀏覽速度
- 簡潔的翻譯彈窗，不遮擋頁面內容

**使用方式：**
1. 在任意網頁上選取想翻譯的文字
2. 選取文字的右上方會出現一個藍色小點
3. 將游標移入藍色小點
4. 翻譯結果立即顯示

**語言設定：**
在插件設定中選擇您的慣用語言，當選取的文字與您的設定語言不同時，將自動翻譯為您的設定語言。
